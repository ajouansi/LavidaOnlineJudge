Installation
=================

$ git clone https://github.com/flrngel/LavidaOnlineJudge.git

$ cd LavidaOnlineJudge

$ mysql -u[USER] -p[PASSWORD] [DATABASE]< schema.sql

$ git submodule init

$ git submodule update
